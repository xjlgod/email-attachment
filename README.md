# Email-Attachment
* 你是否因为内外网开发，每次将外网的东西发送进内网感觉很麻烦？
* 你是否只能通过邮件附件的形式进行发送外网的东西？
* 你是否因为单位邮箱有附件大小限制而每次必须多次发送？

如果是的话，就用它吧，支持发送方批量发送邮件（目前仅仅支持使用qq邮箱发送至其他邮箱）并带有附件。

# How to Use
目前只支持qq邮箱发送至任意邮箱，在使用前请开启qq邮箱的SMTP服务，参考：
